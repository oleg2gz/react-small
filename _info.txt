Task 1
3 buttons in a row: users, posts, comments.
Users button is active by default (current state).
Выбранная кнопка выделяется цветом (цвета бэкграунда и текста свопятся).
Click on a button changes current active button and loads data as a list.
Create components: Form, Button, List & ListItem.
Обратить внимание, что на странице отрисовываются объекты, т.е. их надо предварительно переводить в строки.

Resources
https://jsonplaceholder.typicode.com/posts
https://jsonplaceholder.typicode.com/comments
https://jsonplaceholder.typicode.com/users

Move fetch data logic to useFetch hook.

Task 2
Refactor List component to Table component.
Table should be wrapped in div.table-container for CSS formatting.
Не забываем, что некоторые данные всё ещё будут в объектах (и их неплохо бы из объектов достать рекурсией).

Task 4
Подумать о том, что у таблицы могут быть предустановленные колонки (и у разных запросов это разные колонки, т.е. надо будет сделать выбор из трёх вариантов).
